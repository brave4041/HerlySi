01 POLIMORFISME

Konsep di mana sebuah objek dapat memiliki banyak bentuk. Dalam Dart, polimorfisme memungkinkan variabel bertipe superclass untuk menyimpan instance dari subclass, sehingga mempermudah fleksibilitas kode.

  • Flexibility: Kode dapat bekerja dengan berbagai tipe sekaligus.
  • Extensibility: Mudah menambah tipe baru tanpa mengubah struktur utama.
  • is Operator: Digunakan untuk mengecek tipe data objek.
  • as Operator: Digunakan untuk melakukan casting tipe data.
DART // POLIMORFISME 
class Bentuk {
  void gambar() => print('Menggambar bentuk');
}
class Lingkaran extends Bentuk {
  @override
  void gambar() => print('Menggambar lingkaran');
}
class Persegi extends Bentuk {
  @override
  void gambar() => print('Menggambar persegi');
}
void main() {
  Bentuk bentuk1 = Lingkaran(); 
  Bentuk bentuk2 = Persegi();
  
  bentuk1.gambar(); // Output: Menggambar lingkaran
  bentuk2.gambar(); // Output: Menggambar persegi
}

02 OPERATOR 'is' & 'as'

Digunakan untuk memeriksa tipe data (type checking) dan mengubah tipe data (type casting) secara aman.

DART // OPERATORS 
class Kendaraan {}
class Mobil extends Kendaraan {
  void klakson() => print('Tin!');
}
void main() {
  Kendaraan k = Mobil();
  if (k is Mobil) {
    (k as Mobil).klakson();
  }
  // Safe casting dengan as?
  (k as Mobil?)?.klakson();
}

03 ABSTRAKSI (ABSTRACT CLASS)

Abstract class didefinisikan menggunakan keyword abstract. Bisa memiliki abstract method (tanpa body) dan concrete method (dengan body).

DART // ABSTRACT 
abstract class AlatMusik {
  void mainkan(); // Abstract method
  void info() => print('Ini adalah alat musik');
}
class Gitar extends AlatMusik {
  @override
  void mainkan() => print('Jreng jreng...');
}
void main() {
  var g = Gitar();
  g.info();
  g.mainkan();
}

04 INTERFACE (IMPLEMENTS)

Dart menggunakan keyword implements untuk interface. Berbeda dengan inheritance, implements mewajibkan re-implementasi seluruh fungsi.

DART // INTERFACE 
interface class Flyable {
  void terbang() => print('Terbang...');
}
class Bird implements Flyable {
  @override
  void terbang() => print('Burung terbang tinggi');
}
class Plane implements Flyable {
  @override
  void terbang() => print('Pesawat terbang cepat');
}
void main() {
  Bird().terbang();
  Plane().terbang();
}

05 STATIC MEMBERS

Member yang bersifat statis adalah milik class itu sendiri. Contoh: penggunaan static untuk variabel global atau utilitas murni.

DART // STATIC 
class MathUtils {
  static double pi = 3.14;
  static int add(int x, int y) => x + y;
}
void main() {
  print('Nilai pi : ${MathUtils.pi}');
  print('Result   : ${MathUtils.add(10, 20)}');
}

06 BONUS 1 – LATIHAN PEKERJA

Implementasi abstract class Pekerja dengan 3 subclass (Programmer, Dokter, Guru) dan demonstrasi polimorfisme menggunakan List.

DART // BONUS 1 
abstract class Pekerja {
  void bekerja();
}
class Programmer extends Pekerja {
  @override
  void bekerja() => print('Programmer: Sedang menulis kode Dart.');
}
class Dokter extends Pekerja {
  @override
  void bekerja() => print('Dokter: Sedang memeriksa pasien.');
}
class Guru extends Pekerja {
  @override
  void bekerja() => print('Guru: Sedang mengajar di kelas.');
}
void main() {
  List<Pekerja> daftar = [Programmer(), Dokter(), Guru()];
  for (var p in daftar) p.bekerja();
}

07 BONUS 2 – TABLET (MULTIPLE IMPLEMENTATION)

Pembuatan class Tablet yang mewarisi Komputer serta mengimplementasi interface Kamera dan Telepon.

DART // BONUS 2 
class Komputer {
  void boot() => print('Komputer melakukan booting');
}
abstract class Kamera { void foto(); }
abstract class Telepon { void call(); }

class Tablet extends Komputer implements Kamera, Telepon {
  @override
  void foto() => print('Tablet: Ambil foto dengan resolusi tinggi.');
  @override
  void call() => print('Tablet: Melakukan panggilan video.');
}
void main() {
  var myTab = Tablet();
  myTab.boot();
  myTab.foto();
  myTab.call();
}

08 BONUS 3 – MATH UTILS (COMPLETE SYSTEM)

Pengembangan class MathUtils dengan berbagai static methods untuk kalkulasi faktorial, prima, pembulatan, angka acak, rata-rata, hingga median.

DART // BONUS 3 
import 'dart:math';

class MathUtils {
  static int factorial(int n) => n <= 1 ? 1 : n * factorial(n - 1);
  static bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i <= sqrt(n); i++) if (n % i == 0) return false;
    return true;
  }
  static double round(double val, int places) => double.parse(val.toStringAsFixed(places));
  static int randomInt(int min, int max) => min + Random().nextInt(max - min + 1);
  static double average(List<num> nums) => nums.isEmpty ? 0 : nums.reduce((a, b) => a + b) / nums.length;
  static double median(List<num> nums) {
    if (nums.isEmpty) return 0;
    var s = List<num>.from(nums)..sort();
    int m = s.length ~/ 2;
    return s.length % 2 != 0 ? s[m].toDouble() : (s[m - 1] + s[m]) / 2.0;
  }
}
void main() {
  print('Median [1, 3, 5]: ${MathUtils.median([1, 3, 5])}');
  print('Average [10, 20]: ${MathUtils.average([10, 20])}');
  print('Random [1-100]: ${MathUtils.randomInt(1, 100)}');
}

09 TUGAS – CHALLENGE MEDIA PLAYER

Sistem Media Player menggunakan abstract class dan polimorfisme untuk Audio, Video, dan Streaming Player.

DART // CHALLENGE 
abstract class MediaPlayer {
  void play();
}
class AudioPlayer extends MediaPlayer {
  @override
  void play() => print('Memutar Audio MP3...');
}
class VideoPlayer extends MediaPlayer {
  @override
  void play() => print('Memutar Video MP4...');
}
class StreamingPlayer extends MediaPlayer {
  @override
  void play() => print('Memutar Live Stream...');
}
class Playlist {
  List<MediaPlayer> media = [];
  void playAll() => media.forEach((m) => m.play());
}
void main() {
  var list = Playlist();
  list.media = [AudioPlayer(), VideoPlayer(), StreamingPlayer()];
  list.playAll();
}
