01 KONSEP GENERICS

Generics adalah fitur yang memungkinkan pembuatan komponen (class, function, atau interface) yang dapat bekerja dengan berbagai tipe data tanpa mengorbankan Type Safety.

ANALISIS MANFAAT:
  • Strong Typing: Kesalahan tipe data dideteksi saat kompilasi (Compile-time), bukan saat aplikasi berjalan (Runtime).
  • Code Reusability: Kita tidak perlu membuat class yang sama berulang kali hanya karena tipe datanya berbeda (e.g. IntBox, StringBox).
  • Performance: Menghindari overhead dari type casting yang berlebihan.
STANDARD TYPE VARIABLES:
<T> : Type (General)
<E> : Element (Collections)
<K> : Key (Maps)
<V> : Value (Maps)

02 GENERIC COLLECTIONS

Tanpa kita sadari, kita sudah sering menggunakan Generics saat bekerja dengan List dan Map di Dart.

DART // BUILT-IN GENERICS
// List dengan parameter tipe String
List<String> names = ["Herly", "Afrizal"];

// Map dengan parameter tipe String (Key) dan int (Value)
Map<String, int> scores = {"Dart": 90, "Flutter": 95};

  List<String> names = ["Herly", "Afrizal"];
  Map<String, int> scores = {"Dart": 90};
  print("Names: $names");
}

04 IMPLEMENTASI GENERIC CLASS
VS CODE
DARTPAD

Class Box<T> adalah contoh paling dasar di mana T akan digantikan oleh tipe data asli saat instansiasi.

FILE: libgeneric_class.dart
class Box<T> {
  T? _content;
  Box([this._content]); // Constructor

  void put(T item) => _content = item;
  T? get() => _content;
}
FILE: test_generic_class.dart 
import 'libgeneric_class.dart';
void main() {
  var sBox = Box<String>();
  sBox.put("Herly Afrizal");
  print("Isi Box: ${sBox.get()}");
}

06 TYPE INFERENCE IN GENERICS
VS CODE
DARTPAD

Dart cukup cerdas untuk mengetahui tipe data yang kita gunakan tanpa kita harus menulisnya secara eksplisit (Type Inference).

DART // AUTO DETECT TYPE 
import 'libgeneric_class.dart';
void main() {
  var nameBox = Box("Herly"); // Infer sebagai String
  var numBox = Box(2514);     // Infer sebagai int
  
  print("Box 1: ${nameBox.get()} (${nameBox.get().runtimeType})");
  print("Box 2: ${numBox.get()} (${numBox.get().runtimeType})");
}

07 BOUNDED TYPE PARAMETERS

Membatasi tipe data yang boleh digunakan menggunakan keyword extends. Misalnya hanya menerima tipe angka (num).

dynamic getFirst<T extends Iterable>(T data) => data.first;

08 TUGAS PRAKTIKUM: DATA STORAGE
VS CODE
DARTPAD

Implementasi nyata menggunakan List<T> untuk menyimpan sekumpulan data secara dinamis namun tetap aman.

FILE: libdatastorage.dart
class DataStorage<T> {
  List<T> _storage = [];

  void add(T data) => _storage.add(data);
  
  List<T> getAll() => _storage;
  
  int get length => _storage.length;
}
FILE: test_datastorage.dart 
import 'libdatastorage.dart';

void main() {
  print("--- DEMO DATA STORAGE GENERICS ---");
  
  var userStorage = DataStorage<String>();
  userStorage.add("Admin");
  userStorage.add("Herly");
  
  print("Total Data : ${userStorage.length}");
  print("Daftar User: ${userStorage.getAll()}");
}
